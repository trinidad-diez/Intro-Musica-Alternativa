# README
Este proyecto consiste en una página web dedicada a la **música alternativa** y a sus diferentes *subculturas*. Incluye una página principal con información introductoria, además de secciones específicas sobre **géneros musicales**, **artistas destacados**, *valores culturales* y algunos recursos adicionales como playlists y un formulario de newsletter.

## Qué he hecho
He desarrollado y estructurado el sitio web utilizando **HTML** y **CSS**. Organicé el contenido en distintas secciones, añadí *imágenes representativas*, enlacé subpáginas internas y recursos externos, y diseñé una navegación clara. También trabajé la **coherencia visual** para mantener una estética acordada con la temática alternativa.

## Qué creo que he aprendido
Este proyecto me ha permitido afianzar conocimientos sobre la **creación y organización de sitios web estáticos**. He aprendido a estructurar mejor la información, gestionar múltiples páginas, mantener un estilo uniforme y mejorar la *jerarquía visual* y la *legibilidad*. También reforcé nociones de **accesibilidad** y buenas prácticas en la organización del código.

## Qué me ha parecido
El proyecto me ha resultado **interesante** y *dinámico*. La temática musical permite experimentar con estilos visuales distintos, lo que hace el proceso más creativo. En general, ha sido un trabajo útil para poner en práctica conocimientos y mejorar la soltura en el **diseño web**.

## Mayor dificultad
La mayor dificultad ha sido coordinar todos los elementos del sitio para que funcionaran de manera coherente: enlaces, rutas, imágenes, estilos y subpáginas. También supuso un reto manejar una gran cantidad de contenido sin caer en la *sobrecarga visual*. Aun así, el proceso me ha ayudado a mejorar mi **organización**, la *planificación del contenido* y mi capacidad de **resolución de problemas**.
